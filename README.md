# 启动器-ZMD

一个功能强大、界面美观的应用启动器，为用户提供便捷的应用管理和启动体验。

## 功能特点

### 核心功能
- **应用管理**：集中管理多个应用，一键启动
- **密钥验证**：安全的密钥验证机制，保护应用安全
- **自动更新**：支持应用自动检查和更新
- **界面定制**：支持主题切换，提供美观的用户界面
- **窗口管理**：智能窗口位置记忆，提升用户体验

### 实用工具
- **系统工具**：集成资源监控、进程管理等系统工具
- **快捷工具**：内置计算器、文本编辑器、截图工具等
- **网络工具**：集成网络测速、IP查询等网络工具

### 安全特性
- **反调试**：防止恶意调试和破解
- **反注入**：防止DLL注入攻击
- **自校验**：确保应用完整性

## 系统要求

- **操作系统**：Windows 7/8/8.1/10/11
- **处理器**：Intel Core i3或更高
- **内存**：4GB或更多
- **存储空间**：至少100MB可用空间
- **分辨率**：1024x768或更高

## 安装方法

1. **下载安装包**：从官方网站或指定渠道下载最新版本的安装包
2. **运行安装程序**：双击安装程序，按照提示完成安装
3. **启动应用**：安装完成后，桌面会生成快捷方式，双击启动
4. **输入密钥**：首次启动时，需要输入有效的密钥进行验证

## 使用说明

### 基本操作
1. **启动应用**：在主界面点击对应应用的按钮即可启动
2. **查看公告**：左侧公告栏显示最新公告和教程链接
3. **打开目录**：点击应用按钮旁的文件夹图标，可打开应用所在目录

### 高级功能
1. **主题切换**：根据系统主题自动切换亮色/暗色模式
2. **窗口位置记忆**：系统会自动记忆各窗口的位置和大小
3. **快捷键**：支持自定义快捷键启动应用

### 插件系统（未来版本）
- 支持第三方插件扩展功能
- 可通过插件市场下载和安装插件
- 支持自定义插件开发

## 常见问题

### 1. 启动器无法启动
- **解决方案**：检查系统兼容性，确保操作系统版本符合要求
- **解决方案**：以管理员身份运行启动器
- **解决方案**：检查是否有其他实例正在运行

### 2. 密钥验证失败
- **解决方案**：确保输入的密钥正确无误
- **解决方案**：检查网络连接是否正常
- **解决方案**：联系管理员获取有效的密钥

### 3. 应用启动失败
- **解决方案**：检查应用路径是否正确
- **解决方案**：以管理员身份运行启动器
- **解决方案**：检查应用是否安装正确

### 4. 启动器卡顿
- **解决方案**：关闭不必要的应用程序，释放系统资源
- **解决方案**：检查是否有病毒或恶意软件
- **解决方案**：更新启动器到最新版本
