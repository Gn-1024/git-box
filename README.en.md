# Launcher-ZMD

A powerful and beautiful application launcher that provides users with convenient application management and launch experience.

## Features

### Core Features
- **Application Management**: Centralized management of multiple applications, one-click launch
- **Key Verification**: Secure key verification mechanism to protect application security
- **Auto Update**: Supports automatic application check and update
- **Interface Customization**: Supports theme switching, providing a beautiful user interface
- **Window Management**: Intelligent window position memory, enhancing user experience

### Practical Tools
- **System Tools**: Integrated resource monitoring, process management and other system tools
- **Quick Tools**: Built-in calculator, text editor, screenshot tool, etc.
- **Network Tools**: Integrated network speed test, IP query and other network tools

### Security Features
- **Anti-debugging**: Prevent malicious debugging and cracking
- **Anti-injection**: Prevent DLL injection attacks
- **Self-verification**: Ensure application integrity

## System Requirements

- **Operating System**: Windows 7/8/8.1/10/11
- **Processor**: Intel Core i3 or higher
- **Memory**: 4GB or more
- **Storage**: At least 100MB of available space
- **Resolution**: 1024x768 or higher

## Installation Method

1. **Download Installation Package**: Download the latest version of the installation package from the official website or designated channel
2. **Run Installation Program**: Double-click the installation program and follow the prompts to complete the installation
3. **Start Application**: After installation, a shortcut will be generated on the desktop, double-click to start
4. **Enter Key**: When starting for the first time, you need to enter a valid key for verification

## Usage Instructions

### Basic Operations
1. **Start Application**: Click the corresponding application button on the main interface to start
2. **View Announcements**: The left announcement bar displays the latest announcements and tutorial links
3. **Open Directory**: Click the folder icon next to the application button to open the application's directory

### Advanced Features
1. **Theme Switching**: Automatically switch between light/dark mode according to the system theme
2. **Window Position Memory**: The system will automatically remember the position and size of each window
3. **Shortcuts**: Support custom shortcuts to start applications

### Plugin System (Future Version)
- Support third-party plugin extension functions
- Plugins can be downloaded and installed through the plugin market
- Support custom plugin development

## Common Issues

### 1. Launcher Cannot Start
- **Solution**: Check system compatibility, ensure the operating system version meets the requirements
- **Solution**: Run the launcher as an administrator
- **Solution**: Check if there are other instances running

### 2. Key Verification Failed
- **Solution**: Ensure the entered key is correct
- **Solution**: Check if the network connection is normal
- **Solution**: Contact the administrator to obtain a valid key

### 3. Application Launch Failed
- **Solution**: Check if the application path is correct
- **Solution**: Run the launcher as an administrator
- **Solution**: Check if the application is installed correctly

### 4. Launcher Lag
- **Solution**: Close unnecessary applications to free up system resources
- **Solution**: Check for viruses or malware
- **Solution**: Update the launcher to the latest version
